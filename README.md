# CMD - programming course assignment
CMD which is implemented in c ++ language and supports actions like : ls, ls -s , pwd 
cd (with path) , rm (remove) , mv (move) ,history, mkdir and mkfile.

The objective of this assignment :
  * Design an object-oriented system and gain implementation experience in C++.
  * Learning the rule-of-5 principles.
  * Learning C++ memory management and how to avoid memory leaks.
